# emoji-type

> This is literally just a TypeScript package with a type for every recent emoji.

## Installation

### NPM

```sh
$ npm install --save emoji-type
```

### Yarn

```sh
$ yarn add emoji-type
```

## Usage

### With require

```js
const {Emoji} = require('emoji-type');
```

### With ES6 import

```ts
import {Emoji} from 'emoji-type';
```
